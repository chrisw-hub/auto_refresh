1- Installer Python et Pyzo :

https://www.python.org/ 
https://pyzo.org/start.html (faire juste le Step 1, cf image "Step 1")
Prenez juste la dernière version ça devrait marcher, y pas de case à cocher/décocher lors de l'installation, 
Pour python il vous demande à un moment si vous voulez "disable path lenght limit" ça n'a pas d'importance pour mon programme
Du coup vous pouvez le disable ou l'enable comme vous voulez.

2- Ouvrir le programme avec Pyzo, clicker sur "use this environment" et installer les modules nécessaires :

cf images "Use this environment" et "pip install 1" "pip install 2"

Dans le terminal (le shell) écrire : "pip install --upgrade pip" 
puis écrire "pip install pyautogui keyboard opencv-python" pour installer les modules nécessaires

Etape 3 plus nécessaire ! 3- Mettre les images au bon emplacement et vérifier qu'ils ont le bon nom :

Les images doivent être dans le répertoir Ce PC / -- / Utilisateurs / -- 
Le nom des images doit correspondre au nom utiliser dans le code

cf image "Emplacement des fichiers"


4- Set-up les bons raccourcis clavier sur l'émulateur :

Associer à la touche "q" le bouton "rafraichir ds le shop", associer la touche "s" au bouton sur le bouton "confirmer le refresh"
(majuscule/minuscule n'a pas d'importance)

cf images "Bouton Q" et "Bouton S"


C'est bon le programme est opérationnel !


5-Prise en main


Pour le lancer il faut que Bluestacks (ça doit marcher avec d'autres émulateurs normalement tant qu'on peut set-up les raccourcis c'est bon)
soit en full screen et accessible via un simple alt-tab depuis le programme puis appuyer sur ctrl+e pour lancer le refresh shop (depuis le programme)

Pour le stopper appuyer longtemps sur espace ou bouger la souris vers le coin en haut à gauche de l'écran 

(si vous bougez la souris ça vous affichera pas les bm et mystic acheté en revanche)

(en général il y aura un délai du coup si vous voulez quitter rapidement juste faites un alt-tab ça vous vous remettre
sur la fenêtre du code mais attention le programme tournera toujours du coup il faudra l'arrêter en maintenant espace/bouger souris) 

Attention : ce programme ne marche correctement que si la résolution de l'écran est 1920*1080, si c'est en 2560*1440 use la version que néo a. 

Souvent des problèmes peuvent être causés par les images (le programme n'arrive pas à reconnaître le bouton pour acheter les bm ou les mystics)
Dans ce cas lorsque vous tomber sur des mystics et des bm :
Remplacez vos anciennes images par de nouvelles que vous aurez prises (bien respecter le cadrage des images)
